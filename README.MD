# Mythic Keystone Reminder
## Overview
Announces Mythic and Mythic+ dungeon activities when you join or create a group

## Usage
when you join a group it will put in the chat window what key you are joining for you to go back to. this doesnt use say, but just print.

## Slash commands
```
/ksr help    - Displays commands (help is an optional argument)
/ksr remind  - Display cached dungeon name joined through the premade groups
/ksr test    - Prints test display message
```

## Notes
 - displays keystone when you are apart of a group and the leader creates a group listing
 - only caches when you join via the premade group filter. if you invite everyone. it will not have the info to remind you
